# YSGS Website

Static multipage homepage for YSGS, Yonsei Society of Geoscientific Scrutiny.

## Structure

- `index.html`: home page
- `pages/`: individual pages for news, about, officers, join, activities, and contact
- `data/`: editable content files
- `js/`: shared rendering and interaction logic
- `assets/images/`: recommended location for future YSGS activity photos
- `assets/files/`: recommended location for downloadable news attachments
- `styles.css`: global responsive styling

Most future content edits should happen in `data/*.js` rather than in the HTML files.
